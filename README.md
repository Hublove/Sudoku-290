## How to play
The game follows traditional sudoku rules.

### Game rules
The game follows traditional sudoku rules.\
The game board has nine blocks, each block is divided into nine small grid.You need to fill in the Spaces with Numbers 1-9, so that each number 1-9 appear only once in each row, column, and each blocks.
